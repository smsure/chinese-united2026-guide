# chinese-united2026-guide

本仓库用于归档和发布多个独立的 HTML 页面，旨在以静态页面的形式整理和展示各类信息集合。  
仓库不针对任何特定域名、网站或平台，仅作为通用存档与分发用途。

---

## 项目简介

chinese-united2026-guide 是一个以 HTML 页面为载体的资料归档仓库。  
所有页面均为独立的、自包含的静态文件，可离线浏览或部署到任何静态托管服务。  
本仓库不提供任何形式的在线服务、交互功能或第三方链接跳转。

---

## 目录说明

```
chinese-united2026-guide/
├── index.html          # 仓库首页 / 导航页面
├── pages/              # 存放独立 HTML 页面文件
│   ├── example.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `index.html`：仓库的默认入口页面，可在此处添加页面索引或说明。
- `pages/`：所有独立的 HTML 页面存放于此，每个文件可包含不同的主题或内容集合。
- `assets/`：存放页面共用的样式、脚本、图像等资源文件。
